README:

The softwares that we used to build the project are: 
1. Eclipse Jee
2. Mysql 
3. Sqlyog 
The languages we used are JavaScript for the front-end and Java for backend.

The project folder “Notification” contains all the libraries to build the project. 
To run the project, only database creation is required.

We built it in Eclipse Jee, where we opened the folder "Notification" and run it on the Apache Tomcat Server.
Steps to run:
1. In J2ee, open "Notification" folder.
2. Right-click on the project and click on "Run as server".
